Heart rate monitoring system

Heart rate monitoring system is an IoT platform that allow any user,
from any part of the world, to real- time monitor the health status
of a patient that is in possession of the Photopletysmographic sensor device.

The Photopletysmographic sensor device is based on a RaspberryPi,
which everytime is powered starts to acquire the signal from the
Photopletysmographic (PPG) sensor and offers the folloing services,
that can be triggered from this website:

  • Upload data on ThingSpeak
  • View the PPG signal acquisition in real-time
  • View the ThingSpeak data updated
